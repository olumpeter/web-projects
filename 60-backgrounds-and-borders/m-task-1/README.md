# Backgrounds and borders - Task 1

In this task, we want you to add a <code>background</code>, <code>border</code>, and some simple styling to a page header:
    
1. Give the box a <code>5px black solid border</code>, with rounded corners of <code>10px</code>.
1. Add a background image (use the URL <code>balloons.jpg</code>) and size it so that it covers the box. 
1. Give the <code>&lt;h2&gt;</code> a semi-transparent black background color, and make the text white.
